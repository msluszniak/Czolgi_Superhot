# Czolgi_Superhot
Gra w czołgi typu "Superhot" 2D

Parametry wejściowe:
Należy wpisać rozmiar planszy
Następnie wybrać poziom trudności, spośród dostępnych do wyboru (zapisany zostaje ostatnio kliknięty poziom trudności)
Po wybraniu kliknąć przycisk Start gry

Sterowanie:
Ruchy czołgiem: przyciskami A, W, S, D
Obroty lufy: w lewo przycisk Q, w prawo przycisk E
Strzał: przycisk X
